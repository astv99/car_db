# Used Car Database #
### Web application ###

- *Author*:		Stephen Ham
- *Date*: 		March 2018

------------------
**Description**

A simple web app written in PHP to display a "mock" locally-stored MySQL database of some used cars.

------------------------------------------
**Languages & Tools**

- PHP 5.6.25 / HTML5
- CSS3
- MySQL 5.7.14

------------------------------------------
**Deployment Notes**

Use a local xAMP package and access localhost on port 80.
